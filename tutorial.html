What is the 'ReExpoLearn' kit?

'ReExpoLearn' kit is a beautiful resource built for Android. We have redesigned all its components to make it look flat, minimalist and easy to use. We are proud of this new version of ExpoLearn and the possibilities for easy learning.

'ReExpoLearn' is the reboot of ExpoLearn

How to Use It?

The 'ReExpoLearn' kit is built on top of HTML, so using it will be as simple as using a PDF file. Just select either 'Topics' or 'Exercises' in and main menu, swipe through for the chapter or activity you want and click the button to get started.

We provide all the necessary learning resources. To get going, just select what you want.
